  • MAXInterstitialAdapter

    MAXInterstitialAdapter classes wrap interstitial instances from a third party SDK, providing a common interface that MAXInterstitialAd can use for loading and showing interstitials using a third party’s view layer. Subclasses should override loadAd and showAd, and set the interstitialAd property to ensure MAXInterstitialAd can render the ad properly. Instances should also register a delegate to receive events from the underlying interstitial.

    See more

    Declaration

    Swift

    public class MAXInterstitialAdapter: NSObject
  • MAXAdViewAdapter

    MAXAdapterAdView classes wrap UIView instances from a third party SDK, providing a common interface that MAXAdView can use for rendering ads using a third party’s view layer. Subclasses should override the loadAd method and set the adView property to ensure the MAXAdView can render the ad properly. Instances shold also register a delegate to receive events from the underlying adView.
