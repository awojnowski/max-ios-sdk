MAXAdRequest

public class MAXAdRequest

Core API type that packages all of the parameters needed to request a MAX bid. Contains utility functions for making requests to the MAX ad server.

  • requestAd(_:)

    Begin the ad flow by calling requestAd(), which conducts various server side auctions and other ad logic to determine the ad plan.

    The delegate is called with the ad plan when it is ready, after which point, the plan can be executed whenever an ad needs to be shown. Once the ad is shown, the ad request should be discarded.
