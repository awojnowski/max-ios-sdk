MAXMoPubBannerCustomEvent

open class MAXMoPubBannerCustomEvent: MPBannerCustomEvent, MPBannerCustomEventDelegate, MAXAdViewDelegate

MAXMoPubBannerCustomEvent provides a MoPub custom event for banner ads. MoPub will use this class to hand control back to the MAX SDK when MAX has won in the MoPub waterfall. See the documentation on SSP Integration to ensure you integrate this properly in your waterfall.
